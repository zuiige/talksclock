# DESIGN.md

## 气质与意象

**关键词**：舞台聚光灯、控制室时钟、专注、从容

**具象场景**：TED 演讲台侧的提示器——深色幕布背景中，倒计时数字如琥珀色 LED 柔和发光，远距离仍清晰可辨；时间充裕时数字沉稳静立，接近尾声时数字渐变为警示红并微微脉动，像心跳一样提醒演讲者收尾。

## 视觉策略

- 整体暗色调，模拟舞台/控制室环境，减少演讲者视觉干扰
- 倒计时数字占据视觉中心，远距离可读
- 状态变化通过色彩与动效传达，而非复杂 UI 元素
- 控制按钮在非操作时弱化存在感，聚焦时清晰可触

## 配色方案

| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 背景 | `#0a0a0f` | 舞台暗幕 |
| 前景/主文字 | `#f0f0f5` | 聚光灯下的白 |
| 时间充裕 | `#22c55e` (绿) | 绿灯通行 |
| 时间临近（<20%） | `#f59e0b` (琥珀) | 黄灯警示 |
| 时间紧迫（<10%） | `#ef4444` (红) | 红灯即将结束 |
| 超时闪烁 | `#dc2626` 脉动 | 心跳警报 |
| 卡片/面板 | `#1a1a2e` | 控制台面板 |
| 边框/分割 | `#2a2a3e` | 微弱的环境光 |
| 次要文字 | `#8b8ba0` | 幕布边缘的微光 |

## 字体排版

- 倒计时数字：等宽字体 `JetBrains Mono`，粗体，保证数字等宽不跳动
- UI 文字：系统默认 sans-serif
- 排版节奏：倒计时数字极大（视口 15-25%）、标签/辅助文字小而淡

## 动效与交互

- 时间临近：数字颜色渐变过渡（绿→琥珀→红），无突变
- 时间紧迫：数字微脉动（scale 1.02 + glow），频率随紧迫度加快
- 超时：数字闪烁 + 背景红色脉冲
- 按钮悬停：轻微上浮 + 亮度提升
- 任务切换：淡入淡出过渡
- 进度条：顺滑线性缩减，无跳跃

## 显示样式

1. **经典数字**：大字体等宽数字 + 环形进度条
2. **极简全屏**：仅数字，占据全屏，适合远距离观看
3. **进度条模式**：水平/圆弧进度条 + 数字，信息密度更高

## 设计禁忌

- 禁止花哨的粒子效果或 3D 动画——演讲场景需要专注而非炫技
- 禁止亮色大面积背景——舞台场景暗色调更合适
- 禁止小字号倒计时——核心信息必须远距离可读
- 禁止自动播放声音——需用户主动开启提醒
- 禁止动画影响可读性——脉动幅度必须克制
